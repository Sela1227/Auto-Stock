## 🚀 任務一：市場感知快取優化

### 背景問題

原本系統每次查詢股票都會呼叫 Yahoo Finance API，即使在非交易時間（如台股收盤後），導致：
- 不必要的 API 呼叫
- 回應時間長（1-3 秒）
- 浪費資源和可能觸發 API 限制

### 解決方案

實作智慧快取判斷邏輯：
- **市場開盤中**：正常呼叫 API 更新數據
- **市場已收盤**：直接使用本地快取數據，零 API 呼叫

### 實作檔案

| 檔案 | 路徑 | 說明 |
|-----|------|------|
| `price_cache_service.py` | `app/services/` | 新增智慧快取判斷函數 |
| `stock.py` | `app/routers/` | 修改查詢端點使用智慧快取 |
| `watchlist.py` | `app/routers/` | 新增市場狀態回傳 |
| `crypto.py` | `app/routers/` | 修復加密貨幣快取 |

## 🔧 任務二：圖表功能修復

### 問題 1：時間範圍按鈕無法點擊

#### 現象
點擊 1M、3M、6M、1Y 等按鈕沒有反應，圖表不會切換時間範圍。

#### 原因分析
1. `chartFullscreen` 模板中的按鈕**沒有 `onclick` 事件**
2. 原本依賴 `document.addEventListener('click')` 的事件委派，但被其他 JS 覆蓋

#### 修復方案
```html
<button type="button" onclick="setChartRange(22)" class="chart-range-btn" data-days="22">1M</button>
```

同時使用 **capture 模式**的事件監聽確保最高優先級：
```javascript
document.addEventListener('click', function(e) {
    var btn = e.target.closest('.chart-range-btn');
    if (btn) {
        e.preventDefault();
        e.stopPropagation();
        e.stopImmediatePropagation();
        var days = parseInt(btn.getAttribute('data-days'));
        setChartRange(days);
    }
}, true);  // true = capture 模式
```

### 問題 2：MA 均線不顯示

#### 原因分析
原始 `renderFullscreenChart` 函數直接對 `chartData.ma20` 呼叫 `.slice()`，沒有檢查是否存在。

#### 修復方案

```javascript
// 安全切片函數
function safeSlice(arr, start) {
    if (!arr || !Array.isArray(arr)) return [];
    return arr.slice(start);
}

// 檢查是否有有效數據
function hasValid(arr) {
    if (!arr || arr.length === 0) return false;
    for (var i = 0; i < arr.length; i++) {
        if (arr[i] !== null && arr[i] !== undefined && !isNaN(arr[i])) {
            return true;
        }
    }
    return false;
}
```

### 問題 3：圖例無法點擊切換

#### 修復方案
```javascript
plugins: {
    legend: {
        onClick: function(e, legendItem, legend) {
            var index = legendItem.datasetIndex;
            var ci = legend.chart;
            var meta = ci.getDatasetMeta(index);
            meta.hidden = meta.hidden === null ? !ci.data.datasets[index].hidden : null;
            ci.update();
        }
    }
}
```

### 問題 4：多個 JS 檔案函數覆蓋

#### 修復方案

使用 `Object.defineProperty` 鎖定函數：
```javascript
function install() {
    delete window.setChartRange;
    Object.defineProperty(window, 'setChartRange', {
        value: _setRange,
        writable: false,
        configurable: true
    });
}

// 多次執行確保成功
setTimeout(install, 100);
setTimeout(install, 500);
setTimeout(install, 1000);
```
