## ✨ 新增功能

### 比特幣價格卡片

位置：儀表板頁面，恐懼貪婪指數下方

**功能特色：**
- 即時 BTC 價格（美元）
- 24h 漲跌幅 + 週/月漲跌
- 動態背景色（大漲綠、大跌紅、平盤橘）
- 每 60 秒自動更新
- 點擊跳轉查詢 BTC 詳情

---

## 🔧 整合方式

直接用 `static/dashboard.html` 替換專案中的同名檔案即可。

---

## 📡 API 依賴

```
GET /api/crypto/BTC
```

回應格式：
```json
{
    "success": true,
    "price": { "current": 97000 },
    "change": { "day": 2.5, "week": 5.2, "month": 10.3 }
}
```

## 🎨 UI 說明

| 狀態 | 背景色 |
|------|--------|
| 大漲 (≥3%) | 綠色漸層 |
| 大跌 (≤-3%) | 紅色漸層 |
| 平盤 | 橘黃漸層（預設） |
