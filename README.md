# AutoStock

**SELA 多用戶自動選股分析系統**  
全端 Web App，FastAPI + Vanilla JS + PostgreSQL，部署於 Railway

---

## 版本歷程

### V1.02（2026-04-07）
- 🚀 **極簡排程**：移除所有自動價格更新
- 🚀 **每日預載**：只在美股開盤前（21:00）預載一次
- 🚀 **即時查詢**：用戶查才查，不預先更新價格
- 🚀 **移除管理員自動更新**：登入不再觸發全部刷新
- 排程從 15+ 次/天 降至 **6 次/天**

### V1.01（2026-04-07）
- 情緒指數內存快取：60 秒過期
- 指數資料內存快取：60 秒過期

### V1.0.0（2026-04-07）
- 初始版本正式發布

---

## 排程任務（V1.02）

| 任務 | 時間 | 說明 |
|------|------|------|
| 每日預載 | 21:00 | 情緒 + 指數 + 匯率（美股開盤前）|
| 情緒補充 | 09:00 | 早上補一次 |
| 匯率補充 | 09:30 | 早上補一次 |
| 訂閱源 | 08:00、18:00 | 每天 2 次 |

**總計：6 次/天**（原本 15+ 次）

---

## 查價邏輯

1. **情緒/指數**：優先讀內存快取（60 秒）→ DB → 外部 API
2. **股票價格**：用戶查詢時即時查 Yahoo Finance（有快取判斷）
3. **加密貨幣**：即時查 CoinGecko

---

## 開發規範

- 版本命名：新功能 +0.1、Bug fix +0.01、大改版 +1.0
- 每次發布必須更新 `app/config.py` 的 `APP_VERSION`
