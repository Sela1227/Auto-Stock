## 情緒指數流程

```
排程更新（每天 2 次）          用戶查詢
     │                           │
     ▼                           ▼
┌─────────────┐           ┌─────────────┐
│ 外部 API    │           │ 內存快取    │ ← 60 秒
│ CNN/Alt.me  │           │ (不調用API) │
└──────┬──────┘           └──────┬──────┘
       │                         │ 未命中
       ▼                         ▼
┌─────────────────────────────────────────┐
│            PostgreSQL DB                │
│         market_sentiment 表             │
└─────────────────────────────────────────┘
```

| 時間 | 動作 |
|------|------|
| 21:00 | 每日預載（調用外部 API）|
| 09:00 | 早上補充（調用外部 API）|
| 用戶查詢 | **只讀 DB/快取** |
